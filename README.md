# pet925 | Premium Pet Nutrition Guide

科学的な根拠に基づいたプレミアムペットフードの選定ガイドシステムです。
スプレッドシート（マスターデータ）からWebサイトを自動生成する仕組みを構築しています。

## 🚀 プロジェクトの目的 (Objective)
- 複雑なペットフードのデータをミスなく、楽に管理する。
- A8.netやAmazon等のアフィリエイト報酬を最大化する「最安値比較機能」の提供。
- AI（Gemini）を開発パートナーとして、爆速でコンテンツを拡充する。

## 📂 構成ファイル (Context)
### 📂 Directory Structure
- **`src/`** : `convert.js` (変換プログラム), `convert.test.js` (テスト)
- **`data/`** : `pet925_master.ods` (マスター), `products.csv` (中間データ)
- **`public/`** : `index.html` (画面), `data.js` (配信データ)
- **`docs/`** : `MANUAL.md` (運用手順書), `PROJECT_SUMMARY.md` (開発記録)
- **`README.md`** : プロジェクトの全体概要（このファイル）

## 🛠 開発者コマンド
```bash
# 1. 依存関係のインストール
npm install

# 2. 監視モード（保存するだけで自動更新）
npm start

# 3. テストの実行
npm test
```

## 🤖 AIへの指示 (AI Instructions)
Geminiに相談する際は、以下の情報をコンテキストとして渡してください。
- **役割**: 世界クラスのソフトウェアエンジニアリングアシスタント。
- **ルール**: 
    - データの更新は必ず `products.csv` の列構成（14列）を守ること。
    - タグは `MANUAL.md` にある許可リスト以外は使わないこと。
    - `convert.js` のロジックを壊さないように、解析コードの修正は慎重に行うこと。
