# pet925 プロジェクト・サマリー

## 🏆 到達レベル
**実戦的な開発現場で通用する「ジュニア〜ミドルエンド級エンジニア」**
単にコードを書くだけでなく、運用の効率化、データの安全性、自動化を統合して設計・構築できるレベルに到達しています。

## 🛠 習得・活用した技術
1. **データ駆動型アーキテクチャ (Data-Driven)**
   - 人間が編集する「マスターデータ(.ods)」と、プログラム用の「中間データ(.csv)」、Web用の「配信データ(.js)」を分離するプロの設計思想。
2. **静的サイト生成 (SSG) と自動化**
   - Node.js を使ったビルドパイプラインの構築。
   - `fs.watch` によるファイル変更の自動検知（監視モード）。
3. **データ品質管理 (QA)**
   - 正規表現（Regex）を駆使した高度な CSV パース処理。
   - 許可タグリストに基づく自動バリデーション（入力ミス検知）。
   - **Jest** による自動ユニットテストの導入。
4. **モダン・フロントエンド**
   - Vanilla JS によるデータ駆動の UI 生成。
   - 複数ショップの価格比較アルゴリズムと「最安値」自動判定。
   - CSS Grid/Flexbox や Backdrop-filter を使ったカタログ風デザイン。

## 📝 実施内容の記録

### 1. データ管理の革新
- **マスターファイルの導入**: 許可リストや入力規則が消えないよう `.ods` 形式で原本を管理。
- **アフィリエイト・価格連携**: A8.net、Amazon、楽天、Yahoo!の価格比較列を追加し、収益化の基盤を作成。

### 2. システムのインテグレーション
- **変換スクリプト (`convert.js`)**: CSVを高速に JS へ変換する心臓部を開発。
- **監視モードの搭載**: 保存するだけで即座にサイトへ反映される開発環境を実現。
- **タグの一元管理**: 1箇所の修正で「チェック機能」と「サイトのボタン」が連動する仕組み。

### 3. ユーザー体験 (UX) の向上
- **スマート検索**: 商品名、ブランド、タグ、さらには特典情報からも検索可能。
- **自動リンク生成**: 個別URLがない場合も、商品名で各モールの「安い順検索」へ自動誘導。
- **最安値バッジ**: 3モールの価格から自動で最も安いショップを強調表示。
- **エラー画面の親切化**: 検索結果が0件の際のおしゃれなメッセージ表示。

### 4. 運用のプロフェッショナル化
- **詳細なマニュアル (`MANUAL.md`)**: タグリスト、運用ルーチン、トラブル対処法を完備。
- **プロジェクトドキュメント (`README.md`)**: プロジェクトの目的やAIへの指示書を整理。
- **テストコード (`convert.test.js`)**: ロジックが壊れていないか機械が証明する仕組み。
